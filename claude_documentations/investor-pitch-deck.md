# Investor Pitch Deck

## Overview
Created a professional 6-page investor pitch deck for Speechy as a PDF document.

## Files
- `investor-pitch/pitch-deck.html` — Source HTML with embedded CSS (dark theme, landscape A4)
- `investor-pitch/Speechy-Investor-Pitch-Deck.pdf` — Final PDF output (6 pages, landscape)

## Pages
1. **Cover** — Logo, tagline "Privacy-First Voice Intelligence", key stats (29 languages, 100% local, ~3s, $0 cloud cost)
2. **Problem & Solution** — Privacy crisis, high costs, internet dependency vs. Speechy's local AI, instant paste workflow, professional flexibility. Includes "How it Works" flow diagram.
3. **Market & Competition** — STT market ($3.8B→$10.2B, 14.4% CAGR), On-Device AI market ($10.8B→$75.5B, 27.8% CAGR), competitor comparison table (Dragon, Wispr Flow, Otter.ai, macOS Dictation)
4. **Business Model & Revenue** — 4-tier pricing (Free/$0, Pro/$9.99/mo, Lifetime/$149, Business/$19.99/seat), 3-year MRR projection ($2K→$420K), revenue breakdown donut chart, unit economics (LTV:CAC = 6:1, 92% gross margin)
5. **Roadmap & Target Customers** — 4-phase roadmap (Q2'26→2028), 5 customer segments (Healthcare $2.4B TAM, Legal $1.8B, Media $900M, Enterprise $3.2B, Education $600M)
6. **Investment Ask** — $500K pre-seed, $5M Y3 ARR target, fund allocation breakdown

## Technical Details
- Generated using Chrome headless mode with explicit paper dimensions (11.693" x 8.268" = A4 landscape)
- Dark theme matching Speechy brand (blue #007AFF → purple #AF52DE gradient)
- CSS-based charts (bar charts, donut chart, horizontal bars)
- SVG microphone logo
- Inter font family via Google Fonts
- Platforms: macOS and Windows only (no mobile mentioned)
