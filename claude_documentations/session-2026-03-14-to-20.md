## 4. Media Pause/Resume During Recording (FIX)
**Files:** `desktop/SpeechToText/main.swift` ‚Äî `MediaControlManager` class

- **Bug:** Original AppleScript syntax was invalid, media detection always returned false
- **Old (broken):** `tell application "System Events" to return application "Spotify" is running and player state of application "Spotify" is playing`
- **New (working):** Two-step detection ‚Äî check if app running via System Events, then check player state via direct `tell application "Spotify"`
- Direct `pause`/`play` AppleScript commands instead of media key simulation
- Supports Spotify and Apple Music
- Tracks which player was paused to resume correctly
- **Tested:** Spotify play‚Üípause‚Üíresume cycle verified working

## 8. Universal Hotkey System
**Files:** `desktop/SpeechToText/main.swift` ‚Äî `HotkeyConfig`, `HotkeyManager`, `SlotConfigView`

### Phase 1: Key combinations (Cmd+Shift+T etc.)
- Added `keyCode` field to `HotkeyConfig`
- Event tap listens to `keyDown`, `keyUp`, AND `flagsChanged`
- Modifier-only mode preserved, key+modifier mode added

### Phase 2: Universal shortcut recorder
- Removed 4 individual modifier toggle buttons (‚áß ‚åÉ ‚å• ‚åò)
- Replaced with single "Record" button ‚Äî captures ANY combination user presses
- Uses `ShortcutCombinationRecorderView` (NSView subclass) with `keyDown` + `flagsChanged`
- **Accumulation fix:** Modifiers use bitwise OR so Ctrl+Option both register (was overwriting on release)
- **Capture suppression:** `isCapturingShortcut` flag disables hotkey triggers while recording new shortcut

### Phase 3: Unlimited dynamic slots
- Replaced fixed 4 slots (`slot1`-`slot4`) with dynamic `[HotkeyConfig]` array
- Add/remove slots freely (minimum 1, no maximum)
- Each slot: shortcut, language, mode (push-to-talk/toggle), ESC cancel toggle
- Backward compatible migration from old slot1-4 UserDefaults data
- Accent colors cycle through 12 colors
- **Name field removed** (user found it unnecessary)

### Double-trigger fix
- Added 500ms cooldown after every recording stop
- Centralized stop logic into `stopCurrentRecording()` helper
- Prevents modifier-still-held from immediately re-triggering
